# DiegoStrap

An alternative to Bloxstrap written from scratch, designed for maximum efficiency for users who run Roblox executors on Windows.

## 🚀 Quick Start
1. Download the latest release from the [Releases](./releases) page.
2. Run `DiegoStrap.exe` and follow the setup.
3. Launch Roblox with your preferred executor.
